# GET-YA Видео плеер на html5 от сервиса GET-YA.MXIS.RU
Данный плеер поддерживает mp4 но возможно переделать или доработать, вы можете установить на любой хостинг данный модуль так как для его работы требуется только!
PHP 5.6 + 
Установка распаковать на домен или поддомен и все можно приступать к использованию
Если по каким то параметрам фильтр не подошел то не проблема сделать собственный фильтр!
Данный модуль уже интегрирован с сервисом GET-YA 
Описание параметров
ВашСайт/player/index.php?urls=Тут адрес на видео файл который на яндекс диске
ВашСайт/player/index.php?poster=Тут путь или адрес до постера к видео файлу
Как установить на сайт
<iframe src="ВашСайт/player/index.php?urls=Ссылка на видео с яндекс диск&poster=Путь или ссылка на постер" width="610" height="370" frameborder="0" allowfullscreen></iframe>
